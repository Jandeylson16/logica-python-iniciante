# Lógica em Python – Projetos Iniciais

Este repositório contém exercícios e práticas de lógica de programação desenvolvidos em Python, com foco em fundamentos da programação.

## Conteúdo praticado
- Estruturas condicionais (if/else)
- Funções
- Entrada e saída de dados
- Lógica de decisão

## Objetivo
Desenvolver base sólida em programação e resolução de problemas com Python.

## Observação
Repositório em evolução com novos exercícios sendo adicionados conforme avanço nos estudos.
